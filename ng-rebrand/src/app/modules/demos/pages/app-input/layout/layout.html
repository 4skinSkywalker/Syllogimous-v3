The flexgrid lets you define a flexible grid from 1 to 5 columns, which is sort of ideal for forms.

The flexgrid--no-grow is used if you don't want the elements to grow when the elements has wrapped.

--flexgrid-col-min-width controls the minimum width at which a flexgrid item wraps.

--flexgrid-gap controls the horizontal and vertical space between flexgrid items.
