Getting Started

With Angular as base-layer MultiLayout provides a complete out-of-the-box solution along with a super-handy Command Line Interface!

  1. Install NodeJs from NodeJs Official Page
  2. Open Terminal

  3. Install the Angular CLI

    npm install -g @angular/cli

  4. Install MultiLayout dependencies

    npm install

  5. Run the project in your browser!

    ng serve --open

(In case the browser didn't open on its own then navigate to http://localhost:4200/)
